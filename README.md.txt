
README
Επισκόπηση Έργου
Αυτό το έργο υλοποιεί ένα επιτραπέζιο παιχνίδι με σύστημα γύρων. Περιλαμβάνει γραφικό περιβάλλον χρήστη (GUI), λογική παιχνιδιού και unit tests για τις βασικές μεθόδους.

Απαιτήσεις
Java Development Kit (JDK): Έκδοση 11 ή νεότερη.
IDE (προαιρετικά): IntelliJ IDEA (συνιστάται).
Πρόσθετες βιβλιοθήκες: JUnit 5 (για τα tests).
Οδηγίες Εκτέλεσης
Βήματα για την εκτέλεση του κύριου προγράμματος:
Άνοιγμα του Έργου:

Εισάγετε το έργο στο IntelliJ IDEA.
Βεβαιωθείτε ότι το project είναι ρυθμισμένο με το σωστό JDK.
Εκκίνηση του Προγράμματος:

Ανοίξτε το αρχείο Main.java στον φάκελο src.
Πατήστε το κουμπί Run ή χρησιμοποιήστε το συνδυασμό πλήκτρων Shift + F10.

Εκτέλεση Παιχνιδιού:

Ακολουθήστε τις οδηγίες στο γραφικό περιβάλλον χρήστη για να ξεκινήσετε το παιχνίδι.

Βήματα για την εκτέλεση των Unit Tests:

    Τοποθεσία Test Files:

    Τα αρχεία των tests βρίσκονται στον φάκελο tests.

Εκτέλεση Tests στο IntelliJ IDEA:

Κάντε δεξί κλικ στον φάκελο tests ή σε συγκεκριμένα αρχεία test.
Επιλέξτε Run 'All Tests' ή Run 'SpecificTestName'.


src: Περιέχει τον κώδικα του προγράμματος.
tests: Περιέχει τα αρχεία unit tests.
project_assets: Περιλαμβάνει εικόνες, μουσική και άλλα αρχεία του παιχνιδιού.

Σημειώσεις
Βεβαιωθείτε ότι τα αρχεία εικόνων και μουσικής βρίσκονται στη σωστή διαδρομή (project_assets).
Σε περίπτωση σφαλμάτων, ελέγξτε τις διαδρομές των αρχείων και τις εξαρτήσεις της Java.
